## 为什么选择它？

- **⚡ 3 步出方案**：选窝数 → 加精灵 → 点生成，全程无需计算。
- **🧬 智能唯一依赖处理**：自动识别只能与单一雄性配对的雌性，并锁定配对。
- **🗺️ 可视化窝位布局**：7×7 网格 + 曼哈顿距离约束，支持拖拽微调。
- **📸 一键导出图片**：方案图与位置图均可导出，方便分享。
- **🛡️ 数据本地运行**：纯静态页面，无需后端，宠物列表、兼容表均可自定义 CSV。
- **📦 零依赖部署**：一个 HTML 文件 + 一个 CSV，丢到任何静态服务器即用。

## 快速开始

```bash
git clone https://github.com/yourname/rock-kingdom-breeding-helper.git
cd rock-kingdom-breeding-helper
# 直接用浏览器打开 index.html，或使用任意静态服务
使用流程：

设置精灵窝数量（上限 10，自动保留至少 1 个雄性窝）。

点击“添加雌性精灵”选择需要孵蛋的宠物，数量不超过 窝数-1。

点击“添加雄性精灵”输入你实际拥有的雄性库存。

点击 ✨ 智能推荐配窝方案，立即获得最优雄性分配。

（可选）点击 📍 生成位置图 查看 7×7 放置方案，支持拖拽调整、导出图片。
