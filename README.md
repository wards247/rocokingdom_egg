## ✨ 特性

- **3 步出方案**：设置窝数 → 添加精灵 → 一键生成，全程无需心算。
- **唯一依赖智能识别**：自动锁定只能与单一雄性配对的雌性，避免遗漏。
- **可视化窝位布局**：7×7 网格 + 曼哈顿距离约束（≤2步），支持拖拽微调。
- **杜绝雄性闲置**：算法自动精简多余雄性，确保每一只都发挥价值。
- **一键导出长图**：方案详情与精灵窝位置均可截屏导出，方便分享。
- **完全本地运行**：纯静态页面，无隐私风险；宠物列表、兼容表通过 `egg.csv` 自定义。
- **零依赖部署**：一个 HTML + 一个 CSV，扔到任意静态服务器或本地打开即用。
